# 🧩 Blockchain & Multithreading Linux Exercises

ברוכים הבאים לפרויקט התרגול בלינוקס הכולל שני תרגילים נפרדים — כל אחד עוסק בתחום שונה וכולל מימוש מקיף ב־C++/C וב־Bash, בשילוב ספריות חיצוניות, קבצי Makefile, ותיעוד מפורט.

---

## 📂 מבנה הפרויקט

הרפוזיטורי כולל שתי תיקיות תרגול עיקריות:

### 🔹 [ex1](./ex1) — עיבוד שרשרת בלוקים של ביטקוין

- תרגיל העוסק בשליפה, עיבוד וניתוח של מידע משרשרת הבלוקים של ביטקוין.
- כולל סקריפט Bash להורדת מידע, קוד C++ לעיבוד נתונים, שימוש בספרייה משותפת (`libinfra.so`) ועוד.
- יש להריץ `make` ולאחר מכן את הסקריפט `blockchain1.sh` עם מספר בלוקים כדי לטעון את הנתונים הראשוניים.
- קובצי `.cpp` שונים מספקים פונקציונליות לחיפוש, הדפסה, ייצוא CSV ועוד.

📄 למידע נוסף, הוראות הרצה ודוגמאות — ראה קובץ [`ex1/README.md`](./ex1/README.md)

---

### 🔹 [ex2](./ex2) — סימולציית פיצוח סיסמאות מרובה־תהליכים

- תרגיל המדמה תהליך שבו שרת מצפין סיסמה אקראית ולקוחות (Threads) מנסים לפצח אותה.
- ממומש ב־C תוך שימוש בספריות חיצוניות להצפנה ויצירת נתונים רנדומליים.
- כל סבב כולל שרת ותהליכים מתחרים (Clients), הראשון שפיצח את הסיסמה — מנצח.
- מנגנון הסנכרון מתבצע באמצעות `mutex` ו־`condition variables`.

💡 לפני ההרצה יש להתקין את חבילת `mta-utils-dev-x86_64.deb`.

📄 להסבר מלא, דוגמאות ודגשים טכניים — ראה קובץ [`ex2/README.md`](./ex2/README.md)

---

## 🛠 דרישות כלליות

- מערכת הפעלה מבוססת לינוקס.
- Bash ו־G++/GCC מותקנים.
- ידע בסיסי בהרצה מול טרמינל.
- מומלץ לקרוא כל README בתיקיות `ex1` ו־`ex2` לפני הריצה.

---

## 🧑‍💻 איך להתחיל?

1. בחר תרגיל (`ex1` או `ex2`) לפי תחום העניין.
2. קרא את קובץ ה־README שבתיקיה.
3. התקן תלויות במידת הצורך.
4. הרץ `make` לבניית הקוד.
5. הרץ את הקובץ המתאים לפי ההוראות.
